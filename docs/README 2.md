### Research Overview
The research studies how different user segments (technical vs. non-technical) naturally structure and organize application documentation.

### Phase 0 MVP Scope
- Screen 1: Welcome & Consent
- Screen 2: Participant Intake (4 questions)
- Screen 3: Stimulus Introduction (Todo App)
- Screen 4: Intent Capture (freeform text)
- Screen 7: Reflection (5 questions + conditional email)
- Screen 8: Thank You

- Screen 6: Card Sort
